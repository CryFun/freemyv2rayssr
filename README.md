# freemyv2rayssr
selenium过滑块验证，全自动注册机场账号

## 原理
通过比对像素判断缺口位置

## 效果
![image](https://github.com/wxy1343/freemyv2rayssr/raw/main/%E6%95%88%E6%9E%9C.gif)

# 使用方法
## 1.下载脚本
```bash
$ git clone https://github.com/wxy1343/freemyv2rayssr
```
## 2.安装依赖
```bash
$ pip install -r requirements.txt
```
## 3.运行脚本
```bash
$ python register.py
```
